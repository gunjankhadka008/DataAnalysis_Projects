# Restaurant Operations: Predictive Analytics

## 📊 Project Overview
This project analyzes customer spending patterns to understand the relationship between total bill amounts and group sizes. The final goal was to build a machine learning model capable of predicting the number of people in a group based on their transaction value.

## 🛠️ Tech Stack
* **Language:** Python
* **Data Manipulation:** Pandas
* **Visualization:** Seaborn, Matplotlib
* **Machine Learning:** Scikit-Learn (Logistic Regression)

## 📈 Results
* **Key Insight:** Identified distinct spending distributions between Lunch and Dinner shifts.
* **Model Performance:** Achieved a **71% accuracy** rate in predicting group sizes.

## 📁 File Structure
* `my_first_analysis.ipynb`: The main Jupyter Notebook containing the analysis and ML model.
* `netflix_data.csv`: The dataset used for the analysis.
